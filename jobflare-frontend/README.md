# JobFlare.ai Frontend

Earn Additional Income as a Professional - AI Training Jobs Platform

## Technologies Used

- React 18
- React Router v6
- Vite
- CSS3

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will open at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
JobShift.ai/
├── public/          # Static assets (images, favicon, etc.)
├── src/
│   ├── components/  # React components
│   ├── constants/   # Application constants (jobs data, etc.)
│   ├── App.jsx      # Main application component
│   ├── App.css      # Main application styles
│   ├── index.js     # Application entry point
│   └── index.css    # Global styles
└── package.json
```

## Features

- Responsive design with dark/light theme
- Job exploration and application system
- Multiple professional categories
- Testimonials and benefits sections
- FAQ and legal pages
- Smooth scroll animations
